# Linux User Management Lab
## Objective
Practice managing users and permissions in a Linux system, focusing on access control and system security.
## Tools Used
- Linux
- Terminal
- useradd, userdel, chown, groupdel
## Steps
1. Created a new user:
   "useradd username"
2. Modified file ownership:
   "chown user:group file"
3. Deleted a group:
   "groupdel groupname"
4. Deleted a user:
   "userdel username"
## Security Perspective
User and permission management is essential for system security. Proper configuration helps:
- Control access to files and resources
- Prevent unauthorized privilege escalation
- Maintain system integrity
## What I Learned
- How to manage users and groups in Linux
- Importance of file ownership and permissions
- Basic access control concepts
